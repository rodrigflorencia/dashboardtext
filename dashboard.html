Dashboard interactivo de comentarios (CSV) — sin dependencias

Carga un CSV local y obtén un resumen: sentimiento heurístico (positivo/negativo/neutro), palabras frecuentes, distribución temporal, top usuarios y un explorador con filtros. Todo se procesa en tu navegador (sin servidores externos).
Selecciona un archivo CSV para comenzar. El CSV debe incluir como mínimo estas columnas: comentario, fecha, usuario, likes, publicacion, redsocial. La lectura asume un CSV razonablemente bien formado (con comillas dobles para campos con comas o saltos de línea).
